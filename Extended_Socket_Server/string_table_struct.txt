Сначала идёт размер строки в символах (не в байтах) длиной 2 байта.
Затем идёт строка в UNICODE.
Т.е. размер_в_байтах = размер_в_символах * 2.
Если размер строки превышает 0xffff, поле "размер" устанавливается в 0.
Размер таблицы строк всегда выровнен по байту. Это важно. Без выравнивания,
редакторы ресурсов ругаются.
